# How Search Engines Work

Search engines help users find the most relevant information on the internet.

Every search engine generally follows four stages:

1. Crawling
2. Rendering
3. Indexing
4. Ranking

---

## 1. Crawling

Search engine bots discover pages by following links and reading XML sitemaps.

---

## 2. Rendering

Modern search engines render HTML, CSS, and JavaScript to understand the complete page.

---

## 3. Indexing

Useful content is stored in the search engine's index.

---

## 4. Ranking

When someone searches, search engines rank indexed pages according to hundreds of ranking signals.
